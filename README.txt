1.Сервер поддерживает подключение нескольких клиентов;
2. Каждому подключенному клиенту с периодичностью 2 сек отправляет сообщение;
3. При холостом ходе (нет ни одного подключения) по истечении 10 сек серевер выключается (выход из приложения). То есть сервер выключается сам, если клиентов нет.
4. Пути нигде менять не нужно.
